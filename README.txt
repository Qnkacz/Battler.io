 +How to use GIT with unity
 ++GIT RULES
    1. Every Task should be done in its own branch
    2. Name branch according to the task
    3. Before merging branch to main there must be a Code review/Art Review
    4. Each push should be documented accordingly
 ++How to Get the repo working
    1. Git clone this repository
    2. Open unity hub
    3. Go To projects tab
    4. Click [ADD]
    5. Select the .../Battler.io/Battler.io directory, and click <Enter>
